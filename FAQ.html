فێرکاری گشتی لەسەر چۆنیەتی بەکارهێنان

  • زیادکردنی زمان بۆ بەکارهێنان
    (Add Language)

    هەنگاوی یەکەم (١)

    سەرەتا کە تەختەکلیلی کوردی دادەبەزێنیت تەنیا زمانی ئینگلیزی
    بەشێوەی بنەڕەتی چالاکە بۆ نووسین

    پێویستە زمانەکانی تریش زیاد بکەیت بۆ لیستی بەکارهێنان
    بۆ زیادکردنی زمانێکی زیاتر پەنجەبنێ بە دوگمەی (زیادکردنی زمان)



    پاشان لە لیستی زمانەکاندا یەکێ لەو زمانانە هەڵبژێرە کە ئەتەوێ پێی بنووسیت


  • زیادکردنی شێوازی نووسین
    (Add Layout)

    هەنگاوی دووەم (٢)

    دوای زیادکردنی زمان پێویستە شێوازی نووسینیش دیاریبکەیت
    هەر لە بەشی زماندا پەنجەبنێ بە شێوازی نووسین
    (Layout)

    ئینجا شێوازێک لە لیستەکەدا هەڵبژێرە کە خۆت دەستت پێی ڕاهاتووە پاشان پەنجەبنێ بە دوگمەی (زیادکردن)


  • چۆنیەتی گۆڕینی زمانی تەختەکلیل
    (Changing Language)

    تێبینی‌گرنگ: پێویستە هەنگاوی یەکەم و دووەم بەسەرکەوتوویی ئەنجام بدەیت
    تا بتوانی زمانی کیبۆردەکە بگۆڕیت لەکاتی نووسیندا

    بۆ گۆڕینی زمانی تەختەکلیل بۆ زمانێکی تر
    پەنجەت بە ئاڕاستەی لای ڕاست یان چەپ بجوڵێنە
    یان کەمێک دەست لەسەر دوگمەی بۆشایی ڕابگرە


  • چۆنیەتی نووسینی زیکر و کورتە ئایەت
    (Using Zikr & Short Ayat)

    تێبینی: ئەم زیکرانە لەناو دوگمە سەرەکییەکان دانراون
    بە کەمێک دەست ڕاگرتن لەسەریان دەرەکەون

    بۆ نوسینی ( ﷲ ، ﷻ ، ‌‌اللّٰهُ‌ أَكْبَر‌ )
    دەست لەسەر پیتی ( ژ ) ڕابگرە

    بۆ نوسینی ( ﷴ ، ﷺ )
    دەست لەسەر پیتی ( م ) ڕابگرە

    ( ﷽ ) بۆ نوسینی
    دەست لەسەر پیتی ( ب ) ڕابگرە

    بۆ نوسینی ( إِنْ شَاءَ اللّٰه )
    دەست لەسەر پیتی ( چ یان ڤ یان ع ) ڕابگرە

    بۆ نوسینی ( الْحَمْدُ لِلّٰه )
    دەست لەسەر پیتی ( ح ) ڕابگرە

    بۆ نوسینی ( ‌‌أَسْتَغْفِرُ‌ اللّٰه‌‌ )
    دەست لەسەر پیتی ( خ ) ڕابگرە

    بۆ نوسینی ( ‌‌سُبْحَانَ‌‌ ‌اللّٰه‌‌‌‌ )
    دەست لەسەر پیتی ( ن ) ڕابگرە



    هەروەها لای بەشی خەندەکان (ئیمۆجی) چەندین نزا و کورتە ئایەت دانراوە

  • کاریگەری دەنگ و لەرزین
    (Sound & Vibration Settings)

    تێبینی: پێویستە مۆبایلەکەت لە دۆخی ئاساییدا بێت و بێدەنگ نەکرابێ یان پاوەر سەیڤ چالاک نەبێت
    لەهەندێ مۆبایلدا کاریگەری دەنگی و لەرزین کار ناکات گەر دۆخی تاریک بەکارهاتبێ

    بۆ چالاککردن و بەکارهێنانی کاریگەرییەکانی دەنگ و لەرزین
    بچۆ سەر بەشی تەختەکلیل لە ڕێکخستنەکانی بەرنامەکەدا
    هەر لەهەمان شوێن ئەتوانی ئاستی دەنگ و هێزی لەرزین بگۆڕیت


  • بۆشایی و مەودای نێوان دوگمەکان
    (Button Gap & Space)

    بۆ زیادکردنی ماوە و دانانی بۆشایی زیادە لەنێوان دوگمەکاندا
    بچۆ بەشی تەختەکلیل بۆ گۆڕینی مەودای نێوان دوگمەکان


  • قیاسی تەختەکلیل و فۆنتی نووسین
    (Keyboard Text & Font Size)

    بۆ گۆڕینی قەبارەی تەختەکلیل بەرنامەکە بکەرەوە و بچۆ سەر بەشی (تەختەکلیل)
    دواتر هەڵبژاردنی یەکەم نوسراوە (قەبارەی تەختەکلیل) قیاسێک بەدڵی خۆت دیاریبکە

    هەروەها بۆ گەورەکردن یان بچووککردنەوەی قەبارەی فۆنتی نووسین
    بچۆ سەر هەڵبژاردەی دووەم خواروی قەبارەی تەختەکلیل


  • پیشاندان یان شاردنەوەی هێما زیادەکان
    (Showing Alternative Hints)

    بۆ پیشاندان و گۆڕینی جێی هێما و پیتە لاوەکییەکانی خانەی دووەم
    بەشی زمان بکەرەوە لە ڕێکخستنەکانی بەرنامەکە
    لەوێ ئەتوانی شوێنی هێما و پیتە زیادەکانی خانەی دووەم بگۆڕیت



  • دیاریکردن و سڕینەوەی نووسین بەیەکجار
    (Selecting & Delete Long Paragraph)

    بۆ دیاریکردنی نووسینەکان پەنجەت لەسەر دوگمەی سڕینەوە ڕابگرە
    ئینجا بەبێ ئەوەی پەنجەت لابدەیت بە ئاڕاستەی لای چەپ دەست بجوڵێنە


  • ئاگادارکردنەوە لە کاتی لەکارکەوتن
    (Report Error & Bugs)

    لەکاتی وەستانی لەناکاوی بەرنامەکە تکایە ئەم بەشە فەرامۆش مەکە
    پەنجەبنێ بە لەبەرگرتنەوەی ڕاپۆرتی هەڵە، دواتر پەنجەبنێ بە دوگمەی ناردنی ڕاپۆرت بۆ چارەسەرکردن
    بۆ دۆزینەوەی جۆری کێشەکە و چارەسەرکردن و باشترکردنی بەرنامەکەمان لە وەشانەکانی داهاتوودا
