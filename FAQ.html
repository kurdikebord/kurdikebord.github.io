فێرکاری چۆنیەتی بەکارهێنان

  • زیادکردنی زمان و شێواز
    (Add Language & Layout)

    سەرەتا کە تەختەکلیلی کوردی دادەبەزێنیت
    تەنیا شێوازی (ق ڤ ف غ) کوردی
    لەگەڵ
    ئینگلیزی چالاکن بۆ نووسین (Qwerty)

    بۆ چالاککردنی زمان و شێوازە جیاوازەکانی تر
    پەنجە بنێ بەو زمانە یان شێوازەی
    کە دەستت پێی ڕاهاتووە بۆ کاراکردن و بەکارهێنان



    پاشان ئەتوانی بۆ گۆڕینی شوێنی هەر زمان و شێوازێک
    کەمێ دەستی لەسەر ڕابگریت و ڕایکێشیت بۆ ئاڕاستەکان

    بۆ زانیاری زیاتر سەیری ئەم ڤیدیۆیە بکە
    👇

  • چۆنیەتی گۆڕینی زمانی تەختەکلیل
    (Changing Keyboard Language)

    تێبینی‌گرنگ: پێویستە زیاتر لە یەک زمان و شێوازی جیاواز
    چالاککرابێ تا بتوانی زمانەکان بگۆڕیت لەکاتی نووسیندا


    بۆ نمونە: پەنجەت بە ئاڕاستەی لای ڕاست
    یان دوگمەی بۆشایی ببە بۆ لای ڕاست
    بۆ گۆڕینی زمانەکانی تەختەکلیل

    یان بۆ کردنەوەی بەشی هێما و ژمارەکان
    پەنجەت بە ئاڕاستەی لای چەپ بخشێنە (بجوڵێنە)
    👇

  • گۆڕین و هەڵبژاردنی ڕووکار
    (Changing & Select Theme)

    بۆ گۆڕینی ڕەنگ و شێوەی ڕووکارەکانی تەختەکلیل
    بەرنامەکە بکەرەوە و بچۆ سەر بەشی ڕووکەش

    ئینجا یەکەم هەڵبژاردە بکەرەوە ڕووکارەکانی تەختەکلیل
    لێرەدا کام ڕەنگ و شێوەیەیی بەدڵتە پەنجەی پێدابنێ
    بۆ بەکارهێنان وەک ڕووکاری سەرەکی تەختەکلیلەکە

    هەروەها ئەتوانی بچیتە خوارتر
    بۆ بینینی لیستی سەرجەم ڕووکارەکان


  • چۆنیەتی نووسینی زیکر و کورتە ئایەت
    (Using Zikr & Short Ayat)

    تێبینی: ئەم زیکرانە لەناو دوگمە سەرەکییەکان دانراون
    بە کەمێک دەست ڕاگرتن لەسەریان دەردەکەون

    بۆ نوسینی ( ﷲ ، ﷻ )

    دەست لەسەر پیتی ( ل یان ع ) ڕابگرە

    بۆ نوسینی ( ﷴ ، ﷺ ، رسول الله )

    دەست لەسەر پیتی ( م ) ڕابگرە

    ( ﷽ ) بۆ نوسینی

    دەست لەسەر پیتی ( ب ) ڕابگرە

    بۆ نوسینی هەریەک لە

    ( إِنْ شَاءَ اللّٰه ، مَاشَاءَ اللّٰه )
    ( جَزَاكَ‌ اللّٰهُ ‌خَيْرََا ، بَاْرَكَ‌ اللّٰهُ ‌فِیكْ )

    دەست لەسەر پیتی ( ڵ یان ڤ یان ح ) ڕابگرە

    بۆ نوسینی هەریەک لەم زیکرانە

    ( ‌‌سُبْحَانَ‌‌ ‌اللّٰه ، ‌‌أَسْتَغْفِرُ‌ اللّٰه )
    ( ‌‌اللّٰهُ ‌أَكْبَر ، ‌‌الْحَمْدُ لِلّٰه )

    دەست لەسەر پیتی ( ن ) ڕابگرە

    بۆ نوسینی سەلامی دروستی شەرعی

    ( السَّلَاْمُ عَلَیْکُم ، وَ عَلَيْكُمَ السَّلَاْم)

    دەست لەسەر پیتی ( س ) ڕابگرە

    بۆ نوسینی خوا حافیزی و ماڵئاوایی

    ( خوات لەگەڵ ، خوا بتپارێزێ )

    دەست لەسەر پیتی ( ج یان خ ) ڕابگرە



    هەروەها لای بەشی خەندەکان (ئیمۆجی)
    چەندین نزا و کورتە ئایەتی ناو قورئانی پیرۆز
    لەگەڵ چەند شێوازێکی دروستی سەڵاواتدان
    لەسەر پێغەمبەرمان ﷴ ﷺ دانراوە

  • کاریگەری دەنگ و لەرزین
    (Sound & Vibration Settings)

    تێبینی: پێویستە مۆبایلەکەت لە دۆخی ئاساییدا بێت و بێدەنگ نەکرابێ یان پاوەر سەیڤ چالاک نەبێت
    لەهەندێ مۆبایلدا کاریگەری دەنگی و لەرزین کار ناکات گەر دۆخی تاریک بەکارهاتبێ

    بۆ چالاککردن و بەکارهێنانی کاریگەرییەکانی
    دەنگ و لەرزین
    بچۆ سەر بەشی ڕووکەش لە ڕێکخستنەکانی بەرنامەکەدا
    دواتر بەشی ڕێکخستنی کاریگەرییەکان
    (دەنگ - لەرزین - دۆخی تاریک)


    بکەرەوە و لێرەدا ئەتوانی ئاستی دەنگ و هێزی لەرزین بگۆڕیت


    بۆ زانیاری زیاتر سەیری ئەم ڤیدیۆیە بکە
    👇

  • گۆڕینی ئامرازەکانی سەرەوە و خوارەوە
    (Changing Top & Bottom Row)

    بۆ گۆڕینی شێوازی ئامرازەکانی بەشی سەرەوە یان خوارەوە
    بچۆ بەشی ڕووکەش دواتر ڕێکخستنی زیاتر



    بکەرەوە و لێرەدا ئەتوانی شێوازی بەشی سەرەوە یان خوارەوە بگۆڕیت


    بۆ زانیاری زیاتر سەیری ئەم ڤیدیۆیە بکە
    👇

  • دەستکاریکردنی قیاسی تەختەکلیل
    (Editing Keyboard Size)

    بۆ گۆڕینی قەبارەی تەختەکلیل بەرنامەکە بکەرەوە و بچۆ سەر بەشی (ڕووکەش)

    دواتر هەڵبژاردنی سێیەم نوسراوە (ڕێکخستنی زیاتر)

    لێرەدا ئەتوانی قەبارەی تەختەکلیل بەدڵی خۆت بگۆڕیت
    تێبینی: پێویستە قیاس بە ژمارە دیاریبکەیت

    بۆ گەورەکردن ژمارەکە زیاد بکە
    بۆ بچوککردنەوە کەمی بکەرەوە

    بۆ نمونە:
    ڕێژەی 1.5 قەبارەکەی گەورە ئەکات
    ڕێژەی 0.85 قەبارەکەی بچووک ئەبێتەوە



    بۆ زانیاری زیاتر سەیری ئەم ڤیدیۆیە بکە
    👇

  • پیشاندان یان شاردنەوەی پیتە زیادەکان
    (Show & Hide Alternative Hints)

    بۆ پیشاندان و گۆڕینی شوێنی پیتە لاوەکییەکانی خانەی دووەم
    بەشی ڕووکەش بکەرەوە و بچۆ سەر ڕێکخستنی زیاتر

    پاشان هەڵبژاردەی پێنجەم چالاک بکە
    بۆ پیشاندانی پیتەکانی خانەی دووەم
    یان گۆڕینی شوێنی دەرکەوتنیان لەسەر تەختەکلیل



  • ڕێکخستن و دەستکاریکردنی ئاماژەکان
    (Swipe Gesture Settings)

    لەبەشی ئاماژەکاندا ئەتوانی فرمانی هەر دوگمەیەک
    بگۆڕیت و کردارێکی جیاوازی بۆ دیاری بکەیت

    بۆ نمونە: پەنجەت بە ئاڕاستەی لای ڕاست
    یان دوگمەی بۆشایی ببە بۆ لای ڕاست
    بۆ گۆڕینی زمانەکانی تەختەکلیل

    یان بۆ کردنەوەی بەشی هێما و ژمارەکان
    پەنجەت بە ئاڕاستەی لای چەپ بخشێنە (بجوڵێنە)
    👇

    یان دەست لەسەر دوگمەی ئینتەر ڕاگرە
    فرمانەکانی دەستکاریکردنی نووسینی تێدایە

    Select (دیاریکردن) - Copy (لەبەرگرتنەوە)
    Cut (بڕین) - Paste (دانان)


    هەروەها چەندین فرمانی جیاوازی تریش بەردەستە
    ئەتوانی بەدڵی خۆت بیان گۆڕیت بۆ دوگمەکانی تر


  • ڕێکخستن یان گۆڕینی شێوەی ئیمۆجی
    (Change Emoji Face & Settings)

    بۆ چالاککردن یان شاردنەوە (لابردن)ـی
    هەر بەشێکی خەندەکان یان زیکرەکان

    یەک جار پەنجە بنێ بەو بەشەی ناتەوێ
    دەرکەوێت لەبەشی تایبەتی خەندەکاندا

    یان کەمێک دەستی لەسەر ڕاگرە و ڕایکێشە
    بۆ گواستنەوە و گۆڕینی شوێنەکانیان



    هەروەها گەر وویستت شێوەی ئیمۆجی
    iOS مۆبایلەکەت بگۆڕیت بۆ ئیمۆجی ئایفۆن
    سەیری فێرکارییەکان بکە بەپێی جۆری مۆبایلەکەت

    گۆڕینی شێوەی ئیمۆجی سامسۆنگ (گالاکسی)
    👇


    گۆڕینی شێوەی ئیمۆجی شاومی - ڕێدمی - پۆکۆ
    👇


    گۆڕینی شێوەی ئیمۆجی هواویی - ئۆنەر
    👇


    گۆڕینی شێوەی ئیمۆجی ڕیەڵمی - ئۆپۆ
    👇
