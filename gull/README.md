## مەبەست لە ئاوێنە چییە؟

## بۆچی ئەم ئاوێنە پێویستە؟
- بۆ وەی کە ئەگەر وەشانی ڕاستەقینە نەما، وەشانێکی هاوشێوەی لێ وەمێنێت.
- بۆ وەی کە باری دابەزاندنی کەرەستەکان لەسەر شانی وەشانی ڕاستەقینە کەم ببێتەوە.
