GRID INSTRUCTION:

To create a grid container add the ".row" class to the parent element. The grid uses 12 columns. 
Columns must be place inside rows. Columns automatically stack in mobile view.

Here is a simple code example of a grid with two 50% width columns would look like:

<div class="row">
	<div class="six columns"></div>
	<div class="six columns"></div>
</div>

To create a grid with a 75% width and 25% width columns, use the sample code:

<div class="row">
	<div class="eight columns"></div>
	<div class="four columns"></div>
</div>

The grid also supports nested rows. To use nested rows, simply add an element with ".row"
class inside a column. See sample code below:

<div class="row">
	<div class="eight columns"></div>
	<div class="four columns">
		<div class="row">
			<div class="six columns"></div>
			<div class="six columns"></div>
		</div>
	</div>
</div>
